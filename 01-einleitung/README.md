# Übung: Taschenrechner

Ein Taschenrechner, der die vier Grundrechenarten auf mehrere Zahlen anwendet.

## Ausführen

Installieren von `http-server`:

```bash
npm install http-server -g
```

Starten des HTTP Servers in diesem Verzeichnis:

```bash
http-server
```

oder `npx http-server` bei einem Security-Fehler unter Windows.

Im Browser http://127.0.0.1:8080 aufrufen.
